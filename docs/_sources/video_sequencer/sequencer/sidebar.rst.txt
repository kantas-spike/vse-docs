Sidebar
-------

.. The specific content of the sidebar depends on the selected strip. A Movie strip has other properties than a Sound strip. For an overview of all possible properties per strip; see :ref:`Strip Types <striptypes>`.

Sidebarの具体的な内容は、選択したストリップによって異なります。ムービー ストリップには、サウンド ストリップと比べると、異なるプロパティを持ちます。ストリップごとに考えられるすべてのプロパティの概要については、:ref:`Strip Types <striptypes>` を参照してください。
