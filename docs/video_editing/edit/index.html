2.2. プロジェクトを編集する¶

ビデオ プロジェクトを編集するための、あらゆるケースに適合する単一の最適なワークフローはありません。確かにいくつかの特定の使用例はありますが、たとえば、チュートリアルの編集、結婚式のビデオなど、4 つの基本的なアクティビティの区別に関しては、ある程度の合意が存在します。

  1. Montage

    生の映像から始めて、ストーリーを伝える一貫性のある流暢な一連のストリップを提供する必要があります。クリップは再配置、結合、分割（カット）、トリミングする必要があります。

  2. Effects

    これらは、フェードなどのストリップ間の単純なトランジション エフェクトや、ローリング エンド クレジットなどの本格的なアニメーションにすることができます。

  3. Color grading

    組み立てられたタイムラインは、異なる照明条件下で異なるカメラで撮影された異なるショットで構成されているため、色はそれらの間で大きく異なる場合があります。カラー グレーディングと色補正を使用すると、色の認識を調和させることができます。

  4. Sound

    バックグラウンド ミュージックやナレーションの追加から、特別なサウンド効果の作成、Audacity などのサードパーティ ソフトウェアの使用まで、多岐にわたります。
